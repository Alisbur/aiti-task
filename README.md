## AITI Task

Тестовое задание — приложение для управления товарами с авторизацией, сортировкой, поиском и добавлением товаров.

## Стек технологий

- React 19 + TypeScript
- Vite — сборка
- Tailwind CSS — стилизация
- React Router DOM — маршрутизация
- React Hook Form — валидация форм
- TanStack Query — работа с API
- Zustand — управление состоянием (сортировка, сессия)
- Lucide React — иконки

## Функциональность

### Авторизация
- Валидация полей (обязательность заполнения)
- Обработка ошибок API с выводом под полями
- Сохранение сессии:
  - чекбокс ✅ → токен сохраняется после закрытия браузера
  - чекбокс ❌ → сессия сбрасывается при закрытии вкладки

### Список товаров
- Загрузка данных из API с прогресс-баром
- Соответствие столбцам макета Figma

### Сортировка
- По столбцам (цена, рейтинг и др.)
- По возрастанию / убыванию
- Состояние сортировки сохраняется

### Поиск
- Поиск товаров через API

### Добавление товара
- Форма с полями: Наименование, цена, вендор, артикул
- Toast-уведомление при успешном добавлении
- API-сохранение не предусмотрено (локальное добавление)

### Интерфейс
- Рейтинг ниже 3.5 подсвечивается красным
- Кнопка обновления таблицы
- Иконка с тремя точками — заглушка
